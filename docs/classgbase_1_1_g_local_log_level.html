Detailed Description

Help class to temporary change the current log level within the current scope. The previous log level is restored when the class instance is destroyed.

Example usage:

{
GLocalLogLevel(GLogger::LogLevel::None);
// No logging in this scope.
}
// Logging will return to global log level setting setting here
